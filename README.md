# My Telegram Bot

A Telegram bot that gives users an API key when they join the channel.

## How it works
1. User sends /start to the bot
2. User joins the Telegram channel
3. User clicks "Get My API Key"
4. Bot sends their Telegram UID as the API key
5. User pastes the key in the extension to unlock it

## Setup
- Built with Python and pyTelegramBotAPI
- Hosted on Rail
